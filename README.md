## Single Page Application - Starter App

Single Page Application with ui-router and 4 predefined themes: red, teal, brown, indigo.

## Usage
Clone repository:
```
$ git clone https://github.com/ioanungurean/spa-angular-material.git
```

Install `bower`:
```
$ npm install -g bower
```

Run `bower`:
```
$ bower install
```

## Features:
* AngularJS </br>
* AngularUI Router </br>
* AngularMaterial </br>
* HTML5 </br>
* CSS3 </br>
